Ruby on Rails とは

Ruby on Rails とは

RubyのWebアプリケーション開発用フレームワーク。初期のTwitterなどがこの言語で作られました。

Webアプリケーションってなんだっけ？フレームワークって⋯？

設定より規約

Railsの理念です。「決まったやり方（規約）に従えば、細かい設定はしなくてよい」という考え方になります。

普通の開発では、ファイル名・処理の流れ・構成などを細かく「設定ファイル」で指定しないと動かないことがよくあります。

ただしRailsでは、「名前の付け方」「フォルダの場所」などに決まり（規約）があり、それに従えば自動的に動いてくれます。

MVC モデル

Ruby on Railsは基本的にMVCモデルに則って作られています。

MVC モデルって？

基本構造

「設定より規約」の理念とMVCモデルに則り、Railsには基本のフォルダ構成があります。

ゼロから作ってもいいのですが、アプリ用のフォルダを作成し、そこでRailsのアプリケーション生成コマンドを実行すると、自動的にフォルダ群とファイル群を作成してくれます。

          
          rails generate scaffoled 《任意のモデル名》

試しにArticleというモデル名を指定して、アプリケーション生成コマンドを実行すると、以下のようなファイル群が作成されます。

          
# 以下を実行 
rails generate scaffold Article



# 以下のフォルダとファイルが生成される

my_app/                         ← アプリ全体のルートフォルダ（任意の名前）
├── app/                        ← アプリの中心（MVCがここに集まる）
│   ├── controllers/           ← コントローラのフォルダ
│   │   └── articles_controller.rb
│   │
│   ├── models/                ← モデルのフォルダ
│   │   └── article.rb
│   │
│   ├── views/                ← ビューのフォルダ
│   │   └── articles/
│   │       ├── index.html.erb       ← 一覧表示ページ
│   │       ├── show.html.erb        ← 詳細表示ページ
│   │       ├── new.html.erb         ← 新規作成ページ
│   │       ├── edit.html.erb        ← 編集ページ
│   │       └── _form.html.erb       ← フォーム部分（new/editで共通使用）
│   │
│   ├── helpers/   ←ビュー用の補助メソッドを置くフォルダ
│   │   └── articles_helper.rb
│   │
│   ├── mailers/                     ← メール送信用（未使用なら空のまま）
│   └── jobs/                        ← 非同期ジョブ処理（未使用なら空のまま）
│
├── config/                          ← 設定ファイルいろいろ
│   ├── routes.rb                    ← ルーティングを記載するファイル
│   └── environments/               ← 各環境の設定を置くフォルダ
│
├── db/ ←DB設定を置くフォルダ
│   ├── migrate/
│   │   └── 20240620000000_create_articles.rb
│   └── schema.rb 
│
├── public/                         ← 静的ファイル（画像・CSSなど）置き場
│
├── test/                           ← テストコード群
│   ├── controllers/
│   │   └── articles_controller_test.rb
│   ├── models/
│   │   └── article_test.rb 
│   └── fixtures/
│       └── articles.yml
│
├── bin/                            ← Rails実行関連ファイル
├── log/                            ← ログファイル出力場所
├── tmp/                            ← 一時ファイル（キャッシュなど）
├── storage/                        ← アップロードされたファイルの保存先（Active Storage）
├── Gemfile                         ← 使用するGem（ライブラリ）一覧
├── Gemfile.lock                    ← Gemのバージョン固定ファイル
├── Rakefile                        ← タスク定義ファイル
├── config.ru                       ← Rackサーバ用の起動ファイル
└── README.md                       ← アプリの説明文

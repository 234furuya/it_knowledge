Ruby と Java

Ruby とは

Rubyは、1995年にまつもとゆきひろさんによって開発されたプログラミング言語です。

『人間が読みやすいコード』を目指して作られており、Webアプリケーション開発によく使われています。

共通点

  • 演算子、if、while、forといった基本処理
  • クラス、メソッド、変数といったオブジェクト指向要素

相違点

  • 型を表記しない
  • コンパイル不要
  • mainクラス不要
  • ファイル拡張子（.rb）、実行コマンド（ruby {ファイル名（拡張子ごと）}）

サンプルコード

JavaとRubyでそれぞれ「1から5までの数字を出力するプログラム」を書いて実行する方法は以下になります。

ソースコード

Java

              
// Hello.java
public class Hello {
  public static void main(String[] args) {
    System.out.println("こんにちは、世界！");
      for (int i = 1; i <= 5; i++) {
        System.out.println(i + "回目のループです");
      }
   }
}

Ruby

              
# hello.rb
puts "こんにちは、世界！"
for i in 1..5
  puts "#{i}回目のループです"
end

実行方法

Java

            

javac Hello.java// コンパイル
java Hello// 実行

Ruby


ruby hello.rb# 直接実行
